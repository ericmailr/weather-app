
# Weather App

Weather forecast site that fetches data from OpenWeather API.

## Link

https://ericmailr.github.io/weather-app/

## Description

Search a city by zip code or name to display current weather data obtained from OpenWeather. Toggle between Imperial and Metric units.

This project was completed as part of The Odin Project (https://www.theodinproject.com/courses/javascript/lessons/weather-app?ref=lnav) to practice fetching data from an API and using async/await to implement the use of promises.
